# AeroAssist Pro 🛫
Автоматизация поддержки людей с ОВЗ в аэропорту

Этот проект представляет собой современное, отзывчивое и полностью доступное (WCAG) веб-приложение (Single Page Application, SPA), созданное на базе React + Vite + Tailwind CSS.

---

## 🛠️ Как запустить проект на другом компьютере

Чтобы перенести и открыть этот проект на другом устройстве (компьютере или ноутбуке), нужно выполнить несколько простых шагов:

### Шаг 1: Подготовить файлы для переноса
Вам нужно скопировать всю папку с проектом (`aeroassist-pro`), но чтобы она весила мало (а не сотни мегабайт), перед копированием на флешку или отправкой в архиве **УДАЛИТЕ** внутри неё папку `node_modules`. 
*Эта папка содержит тысячи скачанных библиотек, и на новом компьютере мы скачаем их заново.*

В итоге на флешке должны быть файлы: `src`, `public`, `index.html`, `package.json`, `vite.config.js` и т.д.

### Шаг 2: Установить программу Node.js на новом компьютере
Без неё современные веб-сайты не запускаются для локальной разработки.
1. Зайдите на официальный сайт [nodejs.org](https://nodejs.org/).
2. Скачайте версию **LTS** (Recommended for most users) зеленой кнопкой.
3. Установите её как обычную программу (нажимая «Далее»). Перезагрузите компьютер.

### Шаг 3: Распаковать проект и скачать компоненты
1. Перекиньте папку `aeroassist-pro` на новый компьютер (например, на Рабочий стол).
2. Зайдите внутрь этой папки.
3. Нажмите правой кнопкой мыши по пустому месту в папке и выберите:
   - В Windows 11: **"Открыть в Терминале"** (Open in Terminal).
   - В Windows 10: Удерживая клавишу Shift, нажмите правую кнопку мыши и выберите **"Открыть окно PowerShell здесь"**.
   *(Либо просто откройте командную строку `cmd` и через команду `cd` перейдите в эту папку).*
4. В черном окне введите команду для скачивания библиотек:
   ```bash
   npm install
   ```
   *Подождите около 20–40 секунд, пока скачаются пакеты. После этого в папке снова появится `node_modules`.*

### Шаг 4: Запуск сайта!
Теперь, находясь в том же терминале (или открыв новый внутри папки), просто введите команду запуска:
```bash
npm run dev
```

В консоли появится текст со ссылкой, обычно это:
**`http://localhost:5173/`**

Откройте ваш браузер (Chrome, Edge, Яндекс) и вставьте эту ссылку в адресную строку. Готово! Сайт работает!

---

💡 **Важно**: Пока вы тестируете сайт и кликаете по кнопкам, **черное окно терминала должно быть открыто**. Если вы его закроете — сайт "выключится". Чтобы перезапустить его на следующий день, просто снова откройте терминал в папке и напишите `npm run dev`.
